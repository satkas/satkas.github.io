Generowanie plików w systemie Windows i Linux

Published: Fri 02 December 2016
Updated: Sat 06 February 2021
By Satkas

In Informatyka.

tags: bash windows skrypt linux

Czasami zachodzi potrzeba wygenerowania sobie plików na potrzeby różnych testów. W Windows robię w ten sposób.

Otwieram "Wiersz poleceń" i wpisuję:

for /L %%a in (1,1,100)

do

 fsutil file createnew plik-%%a.txt 0

W Linuksie otwieram konsolę i wpisuję:

for i in $(seq 1 100);do

 touch plik-$i.txt

done
